eval.email_consultants Installation
-----------------------------------

To install eval.email_consultants using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``eval.email_consultants`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        eval.email_consultants

* Re-run buildout, e.g. with:

    $ ./bin/buildout
